#  DataWizardd

##  Présentation du projet
DataWizardd est un projet conçu pour expérimenter et pratiquer les bases de la manipulation de données et du workflow GitHub (fork, branche, commit et pull request).

L’objectif principal est d’apprendre à travailler comme dans un environnement de développement collaboratif, en simulant un vrai cycle de contribution open-source.
